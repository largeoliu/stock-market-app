# 图片资源说明

此目录用于存放小程序所需的图片资源。

## 需要的图片文件：

### 基础图标 (32x32 px)
- `search.png` - 搜索图标
- `clear.png` - 清除图标  
- `delete.png` - 删除图标
- `arrow-right.png` - 右箭头图标
- `heart.png` - 收藏图标（空心）
- `heart-filled.png` - 收藏图标（实心）

### 底部导航图标 (48x48 px)
- `home.png` - 首页图标
- `home-active.png` - 首页图标（选中）
- `search.png` - 搜索图标
- `search-active.png` - 搜索图标（选中）

### 功能图标 (32x32 px)
- `chart.png` - 图表图标
- `time.png` - 时间图标

### 状态图片 (200x200 px)
- `empty.png` - 空状态图片

注意：实际项目中需要设计师提供这些图标，或者使用图标字体库替代。
